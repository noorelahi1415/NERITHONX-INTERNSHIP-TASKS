# Day 2 — SSH Setup & Connection with Gitea

## Objective

Understand Gitea's purpose, generate an SSH keypair, configure SSH for secure Git operations, and successfully connect to and clone a repository from Gitea over SSH.

## Contents

| File | Description |
|---|---|
| `01-what-is-gitea.md` | What Gitea is, its purpose, and real-world use cases |
| `02-ssh-key-generation.md` | ED25519 SSH key generation and registration in Gitea |
| `03-ssh-connection-setup.md` | SSH config setup, connection testing, clone/push workflow |
| `04-problems-faced.md` | Issues encountered during setup and how each was resolved |

## Outcome

Successfully generated an SSH keypair, registered it with a Gitea Cloud account, configured SSH for reliable IPv4-based connections, and cloned/pushed a repository over SSH — resolving host misconfiguration and IPv6 timeout issues along the way.

## Technologies Used

Git, OpenSSH, Gitea Cloud
